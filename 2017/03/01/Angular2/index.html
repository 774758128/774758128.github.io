AngularJS Batarang

第一：安装一个 Chrome 插件：AngularJS Batarang

第二：以 http 服务的形式打开你查看调试的页面

第三：打开控制台，找到 AngularJS 选项卡，切换进去

第四：选择 Enable 启动

第五：进入 Scopes 选项卡查看数据模型成员数据的动态变化

注意：如果在使用的过程打开调试看不到 Scopes 记得通过在左上角的刷新按钮右键点击 清空缓存并硬性重新加载 即可。

JSON 序列化

JSON.stringify

作用：将一个 JavaScript 对象转为标准的 json 格式字符串

JSON.parse

作用：将一个标准的 json 格式字符串转为一个 JavaScript 对象

将对象转为 json 格式字符串叫做序列化，
将 json 格式字符串转为对象叫做反序列化。

Web Storage

window.localStorage

作用：永久存储在自己的本地客户端，只要用户不清除

  • setItem(key, value)
    • key 字符串
    • value 字符串
  • getItem(key)
  • removeItem(key)

window.sessionStorage

作用：会话存储，浏览器关闭，数据自动清除
